# Micron Process Engineering Projects

## Project 1: Semiconductor Process Optimization using DOE
- Used factorial design to optimize temperature, pressure, and time
- Improved yield and reduced variation
- Calculated Cp and Cpk for process capability

## Project 2: Predictive Maintenance using SPC
- Applied control charts to monitor equipment
- Detected out-of-control conditions
- Performed root cause analysis (RCA)

## Tools Used

- Statistical Analysis
